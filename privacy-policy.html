Privacy Policy for N8N_APP

Last Updated: 14 November 2025

1. Information We Collect

N8N_APP may collect the following information based on user workflows:

  • Facebook Page details (Page ID, access tokens)
  • Basic Facebook profile information (if granted)
  • Any data the user configures inside n8n automations

We do not collect passwords or sensitive personal information.

2. How We Use Information

  • Automating Facebook Page posts
  • Executing n8n workflows created by the user
  • Improving app functionality

3. Data Sharing

We do not sell or trade user data. Data is shared only with:

  • Facebook APIs executed by user workflows
  • Authorities if required by law

4. Data Security

We take reasonable technical measures to protect your data. You are responsible for securing your n8n server and access tokens.

5. Data Retention

Data is retained only as long as required for active workflows.

6. User Rights

  • Revoke Facebook permissions anytime
  • Delete workflow data from your n8n instance
  • Request deletion by contacting us
